# Product programs (AWT)

Product **programs** are vertical initiatives (e.g. **5G 流動寬頻**) that span multiple **DT platform** components. AI Web Test owns the program shell (structure, journeys, factory); **ReqIQ** is a composable knowledge brick (sources, requirements, readiness, wiki) — no ReqIQ schema changes required.
