# ReqIQ onboarding (PG-2)

Per-initiative upload checklist generated by:

```bash
cd backend
PYTHONPATH=. python scripts/seed-program-reqiq.py --manifest 5g-mobile-broadband
```

Or use API: `GET /api/v1/programs/{slug}/reqiq-onboarding`

Steps:
1. Upload `source_files` per initiative to ReqIQ workspace (`reqiq_project_id` in manifest)
2. Reindex embeddings
3. Create requirements with matching `capabilityKey`
4. Run readiness before factory test-gen
5. Skip `reference_layers` with `automate: false`
